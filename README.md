# TestWork

Ожидание поведения программы: 
При запуске , программа начинает записывать логи , которые указаны в условии в файл/консоль (указывается в настройках), непрерывно. 

Весь код разделён на секторы (папки), поддерживается очень хорошая структура, выполнены принципы SOLID. 

Сама суть настройки: клиент должен поменять config.json под себя( интервал, куда сохранять и как ). Вся программа работает только для Windows.
